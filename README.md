# WanderWise Travel Recommendations

This project is a travel recommendation website built with HTML, CSS, and
JavaScript. It includes a Home page, About Us page, and Contact Us page with
interactive destination recommendations.

**This project was developed as the final project for the IBM Back-End Development course.**

## Pages

- `index.html` — Home page with interactive recommendations.
- `about.html` — About Us page with mission and story.
- `contact.html` — Contact Us page with an email form.

## How to View

Open `index.html` in your browser, or use GitHub Pages to publish the site.
