This is a travel recommendation website developed exclusively for academic purposes as part of the IBM Back-End Development course on Coursera.

The project demonstrates a small static web application built with HTML, CSS, and JavaScript. It includes multiple pages, interactive destination filters, curated travel recommendations, responsive styling, and a contact form interface.

## Project Overview

**WanderWise** is a simple travel recommendation website designed to help users explore destination ideas based on different travel interests.

The website includes:

* A Home page with personalized travel recommendations
* Interactive category filters for destination types
* Recommendation sections for beaches, temples, and countries
* An About Us page presenting the project mission and values
* A Contact Us page with a basic contact form
* Responsive layout and modern visual styling
* JavaScript behavior for filtering content and handling form submission

## Getting Started

Follow these steps to run the project locally.

## Prerequisites

No package manager, framework, or build tool is required.

You only need a modern web browser, such as:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

## Installation

Clone the repository or download the ZIP:

```bash
git clone https://github.com/le0nardomartins/TravelWebsite-IBM-Coursera.git
```

Navigate to the project folder:

```bash
cd TravelWebsite-IBM-Coursera
```

## Running the Project

Open the main file directly in your browser:

```text
index.html
```

Alternatively, you can use the Live Server extension in Visual Studio Code.

If using Live Server, open the project folder in VS Code, right-click `index.html`, and select:

```text
Open with Live Server
```

## Features

* Static travel recommendation website
* Clean and responsive layout
* Navigation between Home, About Us, and Contact Us pages
* Destination cards with images, descriptions, and category tags
* Interactive filters for:

  * All recommendations
  * Beach destinations
  * Temple destinations
  * Country-based destinations
* Smooth scroll behavior from hero buttons to recommendations
* Contact form with basic JavaScript submission feedback

## Deployment

This project can be deployed using GitHub Pages.

Suggested GitHub Pages setup:

```text
Settings > Pages > Deploy from a branch > main > /root
```

After enabling GitHub Pages, the website will be available through the repository's GitHub Pages URL.
